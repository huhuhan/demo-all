## 简要说明

### 核心代码

- 安全自动化配置：`com.yh.auth.security.autoconfigure`
- 认证、鉴权、过滤器：`com.yh.auth.security.authentication`

### 其他

- 项目可作为starter包使用，参考`YHWebSecurityAutoConfiguration`类

- 项目也可直接运行，测试认证、鉴权功能

- 测试用户，账户【yh】密码任意填

- 测试用户，用户【user】【pwd】
